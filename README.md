# Desarrollo de un sistema de punto de venta.
Esta es una actividad práctica para la asignatura *Desarrollo e Implementación de Sistemas de Información*, impartida por la M.C. Rosa Isabel Reynoso Andrés en el Instituto Tecnológico de Ciudad Altamirano.

# El modelo de datos
Se consideran 5 entidades:
- Clientes
- Productos
- Proveedores
- Ventas
- Detalle de venta

# La interfaz gráfica
Se considera un formulario para cada entidad de la base de datos, pudiendo registrar todos los atributos a través de cajas de texto.
